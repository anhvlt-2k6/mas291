MAS291 - Công thức Thống kê

1. Phân phối (Distributions)

Loại 1: Uniform Distribution

1. Rời rạc (Discrete)

Probability Mass Function (PMF):

\[ P(X = a) = \frac{1}{n} \quad (\text{a bất kỳ}) \]

Mean:

\[ \text{Mean} = E(X) = \mu = \frac{a+b}{2} \]

Variance:

\[ \text{Variance} = V(X) = \sigma^2 = \frac{(b - a + 1)^2 - 1}{12} \]

(Note: 'a' and 'b' usually represent the min/max values, 'n' the number of values = b-a+1)

2. Liên tục (Continuous)

Probability Density Function (PDF):

\[ f(x) = \frac{1}{b-a} \quad \text{for } a \le x \le b \]

Mean:

\[ \text{Mean} = E(X) = \mu = \frac{a+b}{2} \]

Variance:

\[ \text{Variance} = V(X) = \sigma^2 = \frac{(b - a)^2}{12} \]

Loại 2: Exponential Distribution (Liên tục)

Ghi nhớ (Key relation):

\[ \mu = \sigma = \frac{1}{\lambda} \]

Probability Density Function (PDF):

\[ f(x) = \lambda e^{-\lambda x}, \quad x \ge 0 \]

Loại 3: Binomial, Poisson, Normal Distribution

1. Binomial (Rời rạc)

Ghi nhớ (Mean & Std Dev):

\[ \mu = np \] \[ \sigma = \sqrt{np(1-p)} \]

2. Poisson (Rời rạc)

Ghi nhớ (Mean & Std Dev):

\[ \mu = \sigma = \lambda \]

3. Normal (Liên tục)

Ghi nhớ (Parameters):

\(\mu, \sigma\) thường cho sẵn (Parameters \(\mu\) and \(\sigma\) are usually given).

2. Kiểm định giả thuyết thống kê cho 2 mẫu độc lập (Means)

Dạng này chia làm 3 phần, ưu tiên từ trên xuống dưới (Consider these cases in order):

1. Nếu đề cho sẵn \(\sigma_1, \sigma_2\) (Known Variances)

Sampling Error (Margin of Error for CI):

\[ E = Z_{\alpha/2} \sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}} \]

Test Statistic:

\[ Z_0 = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} \]

2. Nếu đề có "suppose that \(\sigma_1 = \sigma_2\)" (Unknown but Equal Variances)

Degrees of Freedom:

\[ df = n_1 + n_2 - 2 \]

Pooled Standard Deviation (Variance):

\[ s_p^2 = \frac{(n_1 - 1)s_1^2 + (n_2 - 1)s_2^2}{df} = \frac{(n_1 - 1)s_1^2 + (n_2 - 1)s_2^2}{n_1 + n_2 - 2} \]

Sampling Error (Margin of Error for CI):

\[ E = t_{\alpha/2, df} \sqrt{\frac{s_p^2}{n_1} + \frac{s_p^2}{n_2}} = t_{\alpha/2, df} \sqrt{s_p^2 \left(\frac{1}{n_1} + \frac{1}{n_2}\right)} \]

Test Statistic:

\[ T_0 = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_p^2}{n_1} + \frac{s_p^2}{n_2}}} = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{s_p^2 \left(\frac{1}{n_1} + \frac{1}{n_2}\right)}} \]

3. Nếu đề không cho gì cả (Unknown and Unequal Variances - Welch's t-test)

Degrees of Freedom (Welch-Satterthwaite):

\[ df = \frac{\left(\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}\right)^2}{\frac{\left(\frac{s_1^2}{n_1}\right)^2}{n_1 - 1} + \frac{\left(\frac{s_2^2}{n_2}\right)^2}{n_2 - 1}} \]

(Round \(df\) down to the nearest integer)

Sampling Error (Margin of Error for CI):

\[ E = t_{\alpha/2, df} \sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}} \]

Test Statistic:

\[ T_0 = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}} \]

3. Kiểm định giả thuyết thống kê cho tỷ lệ (Proportions)

Loại 1: Cho 1 mẫu (One Sample)

Sampling Error (Margin of Error for CI):

\[ E = Z_{\alpha/2} \sqrt{\frac{p(1-p)}{n}} \]

Test Statistic:

\[ Z_0 = \frac{p_0 - p}{\sqrt{\frac{p(1-p)}{n}}} \]

Loại 2: Cho 2 mẫu độc lập (Two Independent Samples)

Shared / Pooled Proportion:

\[ p = \frac{x_1 + x_2}{n_1 + n_2} \]

(\(x_i\) = # successes, \(p_i = x_i/n_i\))

Sampling Error (Margin of Error for CI):

\[ E = Z_{\alpha/2} \sqrt{\frac{p_1(1-p_1)}{n_1} + \frac{p_2(1-p_2)}{n_2}} \]

Test Statistic:

\[ Z_0 = \frac{p_1 - p_2}{\sqrt{p(1-p)\left(\frac{1}{n_1} + \frac{1}{n_2}\right)}} \]
